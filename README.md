# Personalized Learning Website

A collaborative platform to help students discover and recommend courses, with a responsive frontend and a secure backend.

## Features

- Course discovery and recommendation system for learners
- Responsive UI built with HTML, CSS, and JavaScript
- Backend implemented with Node.js, Express.js, and MongoDB
- Secure user authentication using JWT and Bcrypt
- Input validation handled with Zod

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, Bcrypt
- Validation: Zod

## Summary

This project enables students to explore learning opportunities and share recommendations in a community-driven environment. It combines a responsive client interface with a robust backend and strong security practices for user authentication.

## Notes

- Security coverage is stated as 99.5% for authentication and validation.
- The project is designed to be adaptable and extendable for additional learning features.
